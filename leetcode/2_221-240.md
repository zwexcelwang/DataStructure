### 226. 翻转二叉树

翻转一棵二叉树。

**难度系数：Easy**

#### 示例

```markown
输入：

     4
   /   \
  2     7
 / \   / \
1   3 6   9
输出：

     4
   /   \
  7     2
 / \   / \
9   6 3   1
```



#### 解

我应该会的

但是我以为我不会

但是我竟然真的会

```java
class Solution {
    public TreeNode invertTree(TreeNode root) {
        if(root==null) {
            return root;
        }
        TreeNode temp = root.left;
        root.left = root.right;
        root.right = temp;
        invertTree(root.left);
        invertTree(root.right);
        return root;
    }
}
```
